Observability service glyph set

Files:
- `obs-mongodb.svg`
- `obs-redis.svg`
- `obs-qdrant.svg`
- `obs-rag-engine.svg`

Usage notes:
- All glyphs are `24x24`.
- They use `currentColor`, so you can drive state with Tailwind classes like `text-emerald-500`, `text-amber-500`, `text-red-500`, or `text-primary`.
- Intended for the observability view service nodes and related legends/tooltips.

# Guion Tanda 3 — Multi-turn manual (5 convs × 5 turnos)

Total: **25 chats**. Tiempo aprox **6-8 min**.

## Reglas

1. Antes de cada conversación → click **"Nueva conversación"**.
2. Manda los 5 mensajes EN ORDEN, **esperando respuesta entre cada uno**.
3. No edites ni reformules. Copia/pega tal cual.
4. Si el bot pide aclaración o se desvía, sigue con el siguiente mensaje del guion (no te enredes).

---

## Conversación 1 — Cultivo arroz

1. `Hola, tengo cultivo de arroz`
2. `¿Qué dosis de Algarium recomiendan?`
3. `¿Y para 5 hectáreas?`
4. `¿Hay otro producto similar?`
5. `Gracias`

---

## Conversación 2 — Cultivo soya

1. `Buenos días, asesoría para soya`
2. `¿Qué bioestimulante tienen?`
3. `¿Composición exacta?`
4. `¿En qué etapa se aplica?`
5. `Perfecto`

---

## Conversación 3 — Frutales con problema

1. `Tengo frutales con clorosis`
2. `¿Qué recomiendan?`
3. `¿Cuánto cuesta?`
4. `¿Hay distribuidor en Lima?`
5. `Listo`

---

## Conversación 4 — Foliares hortalizas

1. `Productos foliares`
2. `¿Cuáles para hortalizas?`
3. `¿Y para tomate?`
4. `¿Aplicación cada cuántos días?`
5. `Ok gracias`

---

## Conversación 5 — Mix general + edge

1. `Hola`
2. `¿Cuál es su producto estrella?`
3. `¿Para qué cultivo?`
4. `Estoy en Cusco, ¿hay cobertura?`
5. `Listo, te aviso`

---

## Después de terminar las 5 conversaciones

Vuelve a la terminal donde corre el script y presiona `ENTER` para arrancar la Tanda 4.
